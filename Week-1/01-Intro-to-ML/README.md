## Why This Topic First

Before writing a single line of code, you need to understand *what* Machine Learning actually is. ML is not magic - it is a family of algorithms that learn patterns from data instead of being explicitly programmed. This section gives you the mental model for everything that follows: what problems ML solves, how it solves them, and the vocabulary you will use every day for the rest of this course.

---

## What You Will Learn

- **What Machine Learning is** - how it differs from traditional programming (rules → answers vs. data → rules)
- **Types of ML** - Supervised Learning (classification, regression), Unsupervised Learning (clustering, dimensionality reduction), and Reinforcement Learning
- **The ML lifecycle** - data collection → preprocessing → model training → evaluation → deployment
- **What a neural network is** - layers, neurons, weights, biases, and activation functions at a high level

---

## Why It Matters

Every concept in this course - from gradient descent to transformers - builds on the vocabulary and mental models you develop here. Skipping this section means you will be memorizing code without understanding *why* it works. Understanding the "what" and "why" of ML makes the "how" dramatically easier.
